# Third-Party Licenses

This directory contains license texts for third-party components used by
PhyloCRISP.

- `booster`:
  - PhyloCRISP uses a modified version of booster.
  - It is bundled in the GUI distribution and also used by CLI-side code paths.
  - Upstream URL: https://github.com/thekswenson/booster
  - License text: `third_party_licenses/booster/LICENSE`.

- `tqDist`:
  - PhyloCRISP bundles `tqDist` binaries for GUI distributions (Windows, macOS,
    and Linux) built by the PhyloCRISP maintainer from upstream source code.
  - Prebuilt `tqDist` binaries are not redistributed directly from upstream.
  - Official URL: https://www.birc.au.dk/~cstorm/software/tqdist/
  - License texts:
    - `third_party_licenses/tqdist/COPYING`
    - `third_party_licenses/tqdist/COPYING.LESSER`
